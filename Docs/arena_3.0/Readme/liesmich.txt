Arena 3.0, 2010-12-24

   * UCIFilter repariert+verbessert
   * Beenden von Motoren nachdem ein Turnier beendet wurde funktioniert nun
   * Man kann nun einen Motor auch im ICS-Modus abschwächen
   * Hauptvarianten können auch als Varianten gespeichert werden (optional)
   * Knoten/ sek. funktioniert jetzt für Winboard Motoren
   * Neue Option für Kiloknoten/Sekunde anstatt Knoten/Sekunde
   * Startklang nun hörbar
   * Autoplayer repariert
   * Wine Turnierbug behoben: Motoren können jetzt neustarten
   * Wine Schriftartenfehler behoben: Wenn Arial order Courier New nicht verfügbar sind, wähle passende Schriftart, nicht zufällige
   * Wine Hilfeanzeigeprogramm hinzugefügt als eigener Download
   * Wine zeigt die Chess Merida Arena Maske nicht an. Maske deaktiviert in Wine, so daß Grafik nicht gestört werden
   * Wenn keine Schrift "Wingdings" verfügbar (normal in Wine), zeige einfache '+','=' und '-' Zeichen, keine unsinnigen
   * Inaktive Schaltflächen im Turniermodus nun inaktiv, nicht ausgeblendet
   * Auto-Analyse: Zusammenfassender Report beginnt mit 1
   * Auto-Analyse: Fehler entfernt, bei dem Stellungen nicht analysiert wurden
   * Auto-Analyse: Neue Option für Parameterinformationen in Reportdatei
   * In der PGN Ausgabe doppelte Resultate in Klammern entfernt, z.B.: {1-0 Arena adjudication} 1-0
   * Überflüssige "..." nach letztem Zug in PGN entfernt
   * Winboard Optionen für Speicher ("memory"), CPU-Kerne ("cores") und Tablebase-Pfade ("egtpath") nun global setzbar
   * Fehler in Comboboxen in Winboard Optionen erntfernt
   * Option "Große Logos" funktioniert wieder

Arena 1.99beta5,  2008-02-09
